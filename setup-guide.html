セットアップガイド

はじめてClaude Codeを使う方向けの手順書です

🍎 Mac
🪟 Windows
事前確認チェックリスト
macOS 12（Monterey）以上であること
インターネットに接続されていること
Claudeのアカウント（Pro または Max プラン）を持っていること
1
Homebrewのインストール
Macにソフトウェアを簡単に入れるためのツールです。すでにインストール済みの場合はスキップしてください。
💡ターミナルの開き方：⌘ + スペース →「ターミナル」と入力 → Enter
Terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
⚠️インストール中にパスワードを求められたら、Macのログインパスワードを入力してください（画面に文字は表示されませんが入力されています）
✅確認：brew --version を実行してバージョンが表示されればOK
2
Node.jsのインストール
Claude Codeの動作に必要なプログラムです。
Terminal
brew install node
✅確認：node --version を実行して v20.x.x 以上が表示されればOK
3
Claude Codeのインストール
本体をインストールします。
Terminal
npm install -g @anthropic-ai/claude-code
✅確認：claude --version を実行してバージョンが表示されればOK
4
Claudeアカウントと紐づける（ログイン）
Claude.ai のアカウントと連携します。
Terminal
claude
💡ブラウザが自動で開きます。claude.ai のアカウントでログインして認証を完了させてください。完了したらターミナルに戻ります。
5
動作確認
デスクトップで起動してみましょう。
Terminal
cd ~/Desktop
claude
✅「何か手伝えることはありますか？」のような表示が出ればセットアップ完了！
よくあるトラブル
症状対処法
command not found: brewSTEP 1 をやり直す
command not found: claudeターミナルを閉じて再度開き、claude --version を試す
ログイン画面が開かないclaude /login を実行する
バージョンが古いclaude update を実行する

事前確認チェックリスト
Windows 10 または 11 であること
インターネットに接続されていること
Claudeのアカウント（Pro または Max プラン）を持っていること
1
Node.jsのインストール
Claude Codeの動作に必要なプログラムです。
💡ブラウザで nodejs.org/ja にアクセスし、「LTS（推奨版）」をダウンロードして実行。すべてデフォルトのままでOK。インストール後はPCを再起動してください。
💡コマンドプロンプトの開き方：Windowsキー + R → cmd と入力 → Enter
✅確認：node --version を実行して v20.x.x 以上が表示されればOK
2
Claude Codeのインストール
コマンドプロンプトで実行します。
コマンドプロンプト
npm install -g @anthropic-ai/claude-code
✅確認：claude --version を実行してバージョンが表示されればOK
3
Windows Terminal のインストール（推奨）
より使いやすいターミナルです。必須ではありませんが、インストールを推奨します。
💡Microsoft Store を開いて「Windows Terminal」を検索し、「入手」をクリック。
4
Claudeアカウントと紐づける（ログイン）
Claude.ai のアカウントと連携します。
コマンドプロンプト
claude
💡ブラウザが自動で開きます。claude.ai のアカウントでログインして認証を完了させてください。完了したらコマンドプロンプトに戻ります。
5
動作確認
デスクトップで起動してみましょう。
コマンドプロンプト
cd %USERPROFILE%\Desktop
claude
✅「何か手伝えることはありますか？」のような表示が出ればセットアップ完了！
よくあるトラブル
症状対処法
'node' は認識されていませんPCを再起動してからもう一度試す
'claude' は認識されていませんコマンドプロンプトを管理者として実行して npm install -g @anthropic-ai/claude-code を再実行
ログイン画面が開かないclaude /login を実行する
バージョンが古いclaude update を実行する
npmのエラーが出るコマンドプロンプトを右クリック →「管理者として実行」で再度試す
