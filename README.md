# YOHAKU AI Workshop

AIで業務を自動化するワークショップ用リポジトリです。

## 使い方

### 1. このリポジトリをダウンロード
```bash
git clone https://github.com/ymizunobyb-dev/yohaku-ai-workshop.git
cd yohaku-ai-workshop
```

### 2. Claude Codeを起動
```bash
claude
```

起動したら、AIが自動でヒアリングを開始します。
